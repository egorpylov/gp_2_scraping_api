## Структура проекта

```
api/
|- config.py             - Конфигурация эндпоинтов и параметров запросов
|- steam_api_parse.py    - Основной скрипт парсинга Steam API

scraping/
|- steam_scrapper.py     - Скрипт скрапинга страниц Steam (Selenium + Logging)
|- config.json           - Настройки браузера и целевых URL

data_processing/
|- merge_json_to_csv.py  - Сборка итогового CSV из JSON-источников
|- join_datasets.py      - Объединение Steam API данных с SteamSpy
|- filter_dataset.py     - Фильтрация и финальная выборка столбцов

notebooks/
|- EDA.ipynb             - Разведочный анализ данных
```

---

## Зависимости проекта

- `pandas` - работа с табличными данными
- `numpy` - численные операции и работа с массивами
- `httpx` - асинхронные HTTP-запросы к API
- `requests` - синхронные HTTP-запросы
- `python-dotenv` - загрузка переменных окружения из .env
- `loguru` - логирование
- `tenacity` - повторные попытки при ошибках запросов
- `more-itertools` - утилиты для работы с итераторами
- `plotly` - интерактивная визуализация данных
- `selenium` - скраппинг
